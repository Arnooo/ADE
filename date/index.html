AngularJS Filter


AngularJS already has a good filter for dates. We just wrote a simple wrapper filter that displays an empty string for 0 instead of Dec 31, 1969.

  • Date 1350400707: {{ 1350400707*1000 | validDate: 'mediumDate' }}
  • Date 0: {{ 0 | validDate: 'mediumDate' }}


AngularJS Directives


There is a nice library called Datepicker for Bootstrap that uses Twitter Bootstrap to make a popup calendar. We made some substantial modifications to get it to work with the Datejs library and AngularJS. In addition to picking from the popup calendar, you can type in things like "today" or "next week". Try it.

YearDateDate+Time


There is also a directive for making a non-input element editable:

{{date2 | date:'yyyy'}}

{{date2 | date:'MMM d, yyyy'}}

{{date2 | date:'MMM d, yyyy h:mm:ss a'}}
