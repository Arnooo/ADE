AngularJS Directives


A simple directive that makes a text string editable. The typical way that we have seen this done is to have hidden input boxes that toggle to visible when clicked. The important difference here is that the input box is created and destroyed only when you needed, so the DOM is not filled with unused elements. This can be a benefit when you have lots and lots of editable things on the page.





The directive will broadcast a message when the edit starts and when it finishes so that your controllers and models can handle the edit properly.
