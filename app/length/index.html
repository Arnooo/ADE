Length filter


A filter that convert a length value into formatted hours and minutes.

  • Length "360": {{ 360 | length }}
  • Invalid length "abcd": {{ "abcd" | length }}


Length editing directive


A directive to edit a length string. You can enter time in hours, minutes or a combination.



{{ lengthstring | length }}


The directive will broadcast a message when the edit starts and when it finishes so that your controllers and models can handle the edit properly.
