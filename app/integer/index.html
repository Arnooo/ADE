Integer Filter


A filter that displays a number as an integer. This is a wrapper for Angular's built in filter so that we can display invalid inputs better.

  • Integer: {{ 1234 | integer}}
  • Invalid integer "integer": {{ "integer" | integer }}


Integer editing directive


A directive that makes an integer editable (uses the same directive as money, decimal and percent).



{{ int | integer }}


The directive will broadcast a message when the edit starts and when it finishes so that your controllers and models can handle the edit properly.
